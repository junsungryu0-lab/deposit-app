# 예치 현황 관리 프로그램 - 설치 및 실행 안내

이 프로그램은 본인 컴퓨터에서만 실행되는 **개인용 로컬 프로그램**입니다.
인터넷에 올라가지 않고, 데이터는 이 폴더 안의 `data/deposits.json` 파일에만 저장됩니다.

---

## 1. 최초 설치 (한 번만 하면 됨)

### ① Node.js 설치
1. https://nodejs.org 접속
2. "LTS" 버튼 클릭하여 설치 파일 다운로드
3. 다운로드된 파일 실행 → "Next" 계속 눌러서 설치 완료

### ② 이 폴더(deposit-app) 압축 해제
원하는 위치(예: 바탕화면, 문서 폴더)에 압축을 풀어두세요.

### ③ 필요한 부품 설치
1. `deposit-app` 폴더로 이동
2. 폴더 안 빈 공간에서 **Shift + 마우스 우클릭** → "여기서 PowerShell 창 열기" (또는 "터미널에서 열기") 클릭
3. 아래 명령어 입력 후 Enter:

```
npm install
```

3~10초 정도 기다리면 설치가 끝납니다. (인터넷 연결 필요)

---

## 2. 실행하기 (매번 사용할 때)

1. `deposit-app` 폴더에서 위와 같이 터미널(PowerShell) 창 열기
2. 아래 명령어 입력 후 Enter:

```
npm start
```

3. 아래와 같은 화면이 뜨면 성공:
```
========================================
  예치 현황 관리 프로그램이 실행되었습니다.
  아래 주소를 웹 브라우저(크롬 등)에 붙여넣으세요:

  http://localhost:3000

  종료하려면 이 창에서 Ctrl + C 를 누르세요.
========================================
```

4. 크롬, 엣지 등 웹 브라우저를 열고 주소창에 `http://localhost:3000` 입력 → 접속

---

## 3. 종료하기

터미널 창을 그대로 닫거나, 터미널 창에서 `Ctrl + C` 를 누르면 종료됩니다.
다시 쓸 때는 "2. 실행하기" 과정만 다시 하면 됩니다 (npm install은 한 번만 하면 됨).

---

## 4. 데이터 백업

화면 하단의 "백업 파일 다운로드" 링크를 누르면 현재 데이터를 JSON 파일로 받을 수 있습니다.
또는 `data/deposits.json` 파일을 그대로 복사해서 다른 곳(USB, 클라우드 등)에 보관하셔도 됩니다.

**주의:** `data/deposits.json` 파일을 잘못 삭제하면 데이터가 사라집니다. 가끔 백업해두는 것을 권장합니다.

---

## 5. 주의사항

- 이 프로그램은 개인 PC에서만 동작하는 **테스트/개인용 버전**입니다.
- 실제 고객 개인정보(실명, 계좌번호 등)를 입력하기 전에, 회사 보안정책 및 개인정보보호 규정을 반드시 확인하세요.
- 여러 사람이 함께 쓰거나 사내 서버에 올리려면, 별도의 보안 검토와 사내 IT팀 협의가 필요합니다.
